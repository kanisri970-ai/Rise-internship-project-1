# ShopWave — RISE Internship Project 1
## Industry-Oriented Responsive Business Website

---

## 📌 Project Overview

**ShopWave** is a fully responsive e-commerce / retail business website built as part of the RISE Internship Web Development Program by Tamizan Skills.

---

## 🛠️ Tools & Technologies Used

| Tool | Purpose |
|---|---|
| HTML5 | Semantic page structure |
| Tailwind CSS (CDN) | Responsive layout & utility styling |
| JavaScript (Vanilla) | Form validation, filtering, navigation, scroll animations |
| Git | Version control |

---

## 📁 Project Structure

```
shopwave/
├── index.html              → Home page
├── js/
│   └── main.js             → Shared JavaScript (nav, forms, animations)
└── pages/
    ├── about.html          → About Us page
    ├── services.html       → Products & Services page
    └── contact.html        → Contact page
```

## 🎯 Features

### Home Page
- Hero section with stats
- Features strip (Free Shipping, Returns, Secure Payment, 24/7 Support)
- Featured product cards with hover animations
- Customer testimonials
- Newsletter signup with validation
- Scroll-reveal animations

### About Page
- Mission & story section
- Stat counters
- Core values grid
- Team section

### Services / Products Page
- Service highlight cards
- Interactive category filter (All / Footwear / Electronics / Fashion / Home)
- Product grid with badges and cart buttons

### Contact Page
- Contact info cards
- JavaScript-validated contact form (name, email, subject, message)
- Success banner on submit
- FAQ accordion

### Shared
- Sticky responsive navbar with mobile menu
- Back-to-top button
- Smooth scroll
- Active nav link highlighting

---

## 🚀 How to Run

1. Unzip / clone the project
2. Open `index.html` in any modern browser
3. Navigate between pages using the navbar

No build tools, no dependencies to install.

---

## 📦 Deployment

This site is deployment-ready for:
- **GitHub Pages** — push to repo → enable Pages → done
- **Netlify** — drag & drop the folder
- **Vercel** — connect repo and deploy
